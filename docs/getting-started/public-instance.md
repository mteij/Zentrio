The public instance has some limitations to ensure fair usage:

- **Rate Limiting**: 100 requests per 15 minutes per IP
- **Profile Limits**: Maximum 10 profiles per user
- **File Uploads**: 10MB limit for avatars
- **No Custom Domains**: Only available via zentrio.eu
