# Getting Started with Zentrio

Welcome to Zentrio! This guide will help you get up and running quickly, whether you want to try our public instance or host it yourself.

## What is Zentrio?

Zentrio is a profile management system for Stremio Web that allows you to:
- Create multiple profiles with unique Stremio credentials
- Maintain individual watch history and preferences
- Customize your streaming experience
- Access enhanced features like addon management
