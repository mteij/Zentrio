# Getting Started with Zentrio

Welcome to Zentrio! This guide will help you get up and running quickly, whether you want to try our public instance or host it yourself.

## What is Zentrio?

Zentrio is a profile management system for Stremio Web that allows you to:
- Create multiple profiles with unique Stremio credentials
- Maintain individual watch history and preferences
- Customize your streaming experience
- Access enhanced features like addon management

## Choose Your Path

### Try the Public Instance
Perfect for immediate use without any setup:
- **Who it's for**: Users who want to try Zentrio instantly
- **Setup time**: 30 seconds
- **Requirements**: Just a web browser
- **Get started**: [Public Instance Guide](public-instance.md)

### Self-Host Your Instance
Ideal for privacy-conscious users and advanced customization:
- **Who it's for**: Users who want control over their data
- **Setup time**: 10-30 minutes
- **Requirements**: Basic technical knowledge, Docker or Node.js
- **Get started**: [Self-Hosting Guide](self-hosting.md)

### Quick Start
For developers and technically advanced users:
- **Who it's for**: Users comfortable with command line
- **Setup time**: 5-10 minutes
- **Requirements**: Docker or Node.js environment
- **Get started**: [Quick Start Guide](quick-start.md)
