# Zentrio

Profile management for Stremio Web.

## Quick Start

### Try Public Instance
Use Zentrio immediately without setup: [zentrio.eu](https://zentrio.eu)

### Self-Host
Deploy your own instance for full control over data.

[Getting Started Guide](getting-started/)

## Features

- Multiple profiles with unique Stremio credentials
- Individual watch history and preferences
- Customizable streaming experience
- Mobile app support
- Self-hosting options
