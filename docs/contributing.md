# Contributing

## Workflow

1. Fork [mteij/Zentrio](https://github.com/mteij/Zentrio)
2. Create branch: `git checkout -b feature/your-feature`
3. Commit changes
4. Run type checks: `bun run type-check`
5. Submit PR

Ensure you use TypeScript and match existing codebase patterns.
See [Development](/development) for local setup.
