# Contributing

Everyone is welcome to contribute to Zentrio!

## How to Contribute

1. Fork the [repository](https://github.com/mteij/Zentrio)
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Run type checking (`bun run type-check`)
5. Commit with a clear message
6. Submit a Pull Request

## What to Contribute

- **Bug fixes**: Found something broken? PRs welcome!
- **Features**: Check the issues for feature requests or propose your own
- **Documentation**: Improvements to docs are always appreciated
- **UI/UX**: Design improvements and accessibility fixes

## Development Setup

See the [Development](/development) guide for setting up your local environment.

## Code Style

- TypeScript for all source files
- Use existing patterns in the codebase
- Keep components focused and reusable

## Translations

Support for translations is coming soon. Stay tuned!

## Questions?

Open an issue on GitHub if you run into any problems.
