# Profile Management

Manage multiple profiles with unique Stremio credentials.

## Creating Profiles

### Via Web Interface

1. Click "Add Profile" on the main page
2. Enter a profile name
3. Provide your Stremio credentials
4. Configure initial settings
5. Click "Create Profile"

### Profile Settings

Each profile maintains:
- Unique Stremio authentication
- Individual watch history
- Personalized addon configuration
- Custom theme preferences
- Independent settings

## Switching Profiles

### Quick Switch

1. Click the profile selector in the top navigation
2. Choose the profile you want to switch to
3. The interface reloads with the selected profile

### Profile Features

- **Watch History**: Each profile tracks its own viewing history
- **Addons**: Configure different addons per profile
- **Settings**: Individual preferences for each profile
- **Avatars**: Custom avatar images for easy identification

## Managing Profiles

### Edit Profile

1. Select the profile you want to edit
2. Click "Settings" or the edit icon
3. Update the desired information
4. Save changes

### Delete Profile

1. Select the profile to delete
2. Click "Delete Profile"
3. Confirm the deletion
4. All data for that profile will be permanently removed

## Profile Data

### What's Stored

- Stremio authentication tokens (encrypted)
- Watch history and progress
- Addon configurations
- UI preferences
- Custom settings

### Data Privacy

- All data is encrypted at rest
- Authentication tokens are securely stored
- No tracking or analytics
- Self-hosted instances keep data on your server

## Best Practices

### Profile Organization

- Use descriptive names for each profile
- Assign unique avatars for visual identification
- Group profiles by user or content type

### Security

- Use strong Stremio passwords
- Regularly update authentication
- Don't share profile credentials

### Performance

- Limit to 10 active profiles per user
- Regularly clean unused profiles
- Clear cache if experiencing issues

## Troubleshooting

### Profile Not Loading

1. Check internet connection
2. Verify Stremio credentials
3. Clear browser cache
4. Try re-authenticating

### Authentication Issues

1. Verify Stremio account is active
2. Check password is correct
3. Ensure Stremio servers are accessible
4. Contact support if issues persist

### Sync Issues

1. Check server connectivity
2. Verify profile settings
3. Refresh the page
4. Try switching profiles back and forth

For more help, see [Troubleshooting](troubleshooting.md) or [Settings](settings.md).
