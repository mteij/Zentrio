# Accounts and Profiles

Zentrio separates the account from the profile.

## Account

An account is used to sign in to a Zentrio server. It owns:

- authentication
- linked providers such as SSO or Trakt
- server-side settings and permissions

## Profiles

Each account can have multiple profiles. A profile keeps its own:

- continue watching
- content filters
- streaming preferences
- addon configuration

Profiles are the main way to separate usage for different people on the same server.

## Typical Setup

1. Create the first account on the server.
2. Add one or more profiles under that account.
3. Adjust profile-specific filters and playback preferences.
4. Install or reorder addons for the profile that will use them.
