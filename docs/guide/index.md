# User Guide

This section is for people using an existing Zentrio server.

## Start Here

1. Read [Accounts and Profiles](/guide/accounts-and-profiles) to understand how users are organized.
2. Read [Addons and Playback](/guide/addons-and-playback) before tuning catalogs, streams, and subtitles.
3. Read [Clients](/guide/clients) when connecting desktop, mobile, or TV devices.

## What Zentrio Does

Zentrio provides:

- account-based access to a server
- multiple profiles per account
- Stremio-compatible addons
- web and native clients
- optional sync and native downloads on supported clients
