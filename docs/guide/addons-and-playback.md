# Addons and Playback

Zentrio uses addons for catalogs, metadata, streams, and subtitles.

## Addons

Zentrio includes a built-in TMDB addon and also supports third-party Stremio-compatible addons by manifest URL.

Addons are configured per profile, which means different profiles can use different addon setups.

## Playback

The player can:

- rank streams
- load subtitles
- remember progress
- show intro, recap, and outro skip buttons when timing data is available

## Practical Advice

- Start with the built-in TMDB addon before adding third-party manifests.
- Add third-party addons one at a time so problems are easier to trace.
- Keep profile filters and addon choices aligned so users see the content they expect.
